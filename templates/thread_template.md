# Thread Template

## Basic Format
```markdown
[_dp] Pulse #[number]
📍 Topic: [Main Theme]

1/5: [Overview + Hook]
#_dp

2/5: Key Points:
💡 [Point 1]
📊 [Point 2]
🎯 [Point 3]
#DataPulse

3/5: Data/Stats:
📊 [Stat 1]
📈 [Stat 2]
📉 [Stat 3]

4/5: Future Implications:
🔮 [Implication 1]
💡 [Implication 2]
⚡️ [Implication 3]

5/5: Source: [Content Title]
via [Platform/Channel]
🔗 [Source Link]
#[Topic] #[Relevant] #[Tags]
```

## Character Limits
- Each section should be ≤ 260 characters
- Include spaces for readability
- Count hashtags in character limit
